# MOONDROP Pudding 实测发现

## 设备信息
- 型号: MOONDROP Pudding (MD-TWS-056)
- 芯片: Zhuhai Jieli (杰理) VID=0x05D6, PID=0x000A
- 蓝牙名: MOONDROP Pudding / TWS-056-BLE (BLE)
- 固件版本: 3.5.2
- MAC 地址: D2:D1:13:AB:67:2

## 协议格式

### SPP (GAIA 格式)
```
TX/RX: FF 04 [Len:2 BE] [Seq:1] [Vendor:1] [FeatureID:1] [CmdID:1] [Payload...]
Len = 1 + 1 + payload.size (feature + cmd + payload)
总包长 = 8 + payload.size
```

### 响应格式
- QUERY 响应: payload=[当前值] (1字节)
- SET 响应: payload=[当前值, ?, ?] (3字节，[1][2]未知)

### BLE (RCSP 协议)
- L2CAP CID: 0xC3FF-0xFFFF (动态)
- 数据以 `00 19` 开头
- 不是标准 GATT，是杰理私有协议
- GATT UUID (OTA): `0000ae00/ae01/ae02`

## SPP 已确认功能

| 功能 | Feature | Cmd | 结果 |
|------|---------|-----|------|
| 固件版本 | 0x05 | 0x00 | ✓ "3.5.2" |
| 序列号 | 0x14 | 0x01 | ✓ ASCII |
| 设备ID | 0x15 | 0x00 | ✓ ASCII |
| EQ状态 | 0x07 | 0x00 | ✓ 空payload |
| 配置查询 | 0x0C | 各cmd | ✓ 配置数据 |
| 设备状态 | 0x0D | 0x07 | ✓ [07,01] |
| Gain查询 | 0x1E | 0x01 | ✓ 增益级别 |
| Gain设置 | 0x1E | 0x02 | ✓ 确认 |
| 支持的命令 | 0x01 | 任意 | ✓ 命令列表 |
| **ANC 查询** | **0x40** | **0x03** | **✓ payload=[模式]** |
| **ANC 设置** | **0x40** | **0x04** | **✓ 确认** |
| **ANC 可用模式** | **0x40** | **0x29** | **✓ [1,1,1,1,1]** |
| 电量 | 0x00 | 0x01 | ✗ 无响应 |
| 编解码器 | 0x0A | - | ✗ 无响应 |

## ANC 模式 (实测确认)

| 值 | 模式 | 状态 |
|----|------|------|
| 0x00 | 关闭 | ✓ 可用 |
| 0x02 | 通透 | ✓ 可用 |
| 0x04 | 降噪 | ✓ 可用 |
| 0x08 | 自适应 | ✗ 暂不可用 |
| 0x10 | 抗风噪 | ✗ 暂不可用 |

## Gain 级别 (实测确认)

| 值 | 级别 |
|----|------|
| 0x00 | 高 |
| 0x01 | 中 |
| 0x02 | 低 |

## BLE 分析 (btsnoop 日志)
- 988 个 BLE 包 (218 TX, 770 RX)
- 所有包以 `00 16` 开头 (RCSP 协议头)
- L2CAP CID: 0xC3FF-0xFFFF
- 无标准 GAIA 格式 (FF 04)
- 无标准 GATT 格式

## 杰理 BLE GATT (OTA 用)
- Service: `0000ae00-0000-1000-8000-00805f9b34fb`
- Write: `0000ae01` (write without response)
- Notify: `0000ae02` (notify)
